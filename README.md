# IMDb 2024 Movie Data Analysis

## Overview
This project scrapes movie data from IMDb for the year 2024 and analyzes it using Python, SQL, and Streamlit.

## Technologies Used
Python
Selenium
Pandas
MySQL
Streamlit
Matplotlib
Seaborn

## Project Workflow
1. Scrape IMDb movie data
2. Clean and preprocess the dataset
3. Store the data in MySQL database
4. Visualize insights using Streamlit dashboard

## Features
- Top rated movies
- Rating distribution
- Votes vs ratings analysis
- Interactive filtering

## How to Run
pip install -r requirements.txt
streamlit run app/streamlit_app.py
